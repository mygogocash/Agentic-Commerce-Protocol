# Welcome to the Agentic Commerce Protocol (ACP) Wiki

**Agentic Commerce Protocol (ACP)** is a next-generation e-commerce backend built on **pure Firebase** architecture. It is designed to be serverless, scalable, and AI-ready, enabling agentic access to product data, user profiles, and cashback rewards.
